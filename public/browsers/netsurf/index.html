NetSurf framebuffer WASM probe

loading nsfb.js

This is a reproducible NetSurf WASM artifact from ports/netsurf/. It is the framebuffer frontend linked against libnsfb's RAM surface, so it does not yet paint to a canvas. A successor should replace the RAM surface with an SDL2/canvas or custom JavaScript surface.

Command line passed to NetSurf: -f ram -w 640 -h 480 about:blank
