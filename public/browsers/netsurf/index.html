NetSurf framebuffer WASM

This page runs the offline full NetSurf framebuffer frontend compiled to WASM with a small libnsfb Emscripten surface. NetSurf's own framebuffer update callback copies only dirty rectangles into Canvas ImageData, and canvas mouse/keyboard events are queued back into fbtk/libnsfb. Networking is intentionally disabled; future socket work should consume BrowserPortWisp from the host app rather than baking an endpoint into the WASM artifact. NetSurf's framebuffer resources are embedded in the Emscripten filesystem at /netsurf (Messages, Choices, CSS, welcome/credits/licence pages, and core icons), so the startup path can load English translations and the offline about page without network access. The current command line is -f emscripten -w 640 -h 480 about:welcome.
