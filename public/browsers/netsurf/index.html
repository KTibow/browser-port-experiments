NetSurf framebuffer WASM

This page runs the offline NetSurf framebuffer frontend compiled to WASM and copies the live nsfb_t RAM framebuffer into Canvas ImageData. This is the full NetSurf framebuffer frontend, not the earlier standalone canvas probe. Networking is intentionally disabled; future socket work should consume BrowserPortWisp from the host app rather than baking an endpoint into the WASM artifact. The current command line is -f ram -w 640 -h 480 about:blank.
