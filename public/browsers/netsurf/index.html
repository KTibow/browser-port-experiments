NetSurf framebuffer canvas probe

This page now displays pixels from a wasm-built libnsfb RAM surface in an HTML canvas. It is still an offline probe: the full NetSurf framebuffer executable (nsfb.js) is checked in, but the browser page uses a small exported libnsfb surface harness until NetSurf's framebuffer frontend exposes its live surface or gains a dedicated Emscripten surface.

Full NetSurf command line currently used for the non-visible executable: -f ram -w 640 -h 480 about:blank.
